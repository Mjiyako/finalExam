C++ Game

Project 3: 4bit Binary Quiz

Arduino Game

This is a game built using an arduino, where the players has to try and find the base 10 equavalent for binary number in set amount of period, I made this game using a circuit board wires, resisitors, LED's, Arduino and a number pad, this was for a project done in a wits university course microprocessors
